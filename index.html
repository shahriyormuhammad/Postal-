# Артикул ДЛ КРП Вклад в ИРП До 60%

Детали по артикулам и размерам

Полный раскрой как в кабинете: по каждому артикулу и размеру — заказы, остаток, оборачиваемость, ДЛ, КТР, КРП и вклад в ИЛ/ИРП. Сразу видно, какой размер какого артикула больше всего тянет ИРП.

Загрузить данные по размерам

CSV: nmId,size,orders,stock,dl — где dl это Доля Локализации размера в процентах (из кабинета WB).

План поставки

Выбери приоритетный артикул — система покажет регионы без покрытия и рекомендуемые склады WB.

Создать поставку

Введите артикул и ростовку (размер × шт в 1 коробке). При повторном вводе того же Vendor Code ростовка подтянется автоматически.

Ростовка (размер × шт в 1 коробке)

Накладная

Вбейте пары «артикул → количество коробок» из бумажной накладной. Система найдёт сохранённую ростовку по артикулу — даже если вы напишете только часть (например, 519-5 найдёт сохранённую А519-5 ТН-10) — и умножит ростовку на указанное количество коробок. Поддерживается сокращённая запись: строка вида -5 читается как «та же база, что в предыдущей строке, но с суффиксом -5» (после А519-2 даст А519-5). Enter в поле количества добавляет новую строку.

Мониторинг прогресса

Окно ИРП — 13 недель скользящее. Первые улучшения видны через 2–3 недели.

Снимки ИРП

Каждое обновление данных сохраняется как снимок. Ниже — динамика.

ДЛ по приоритетным артикулам

Перераспределение

Ежедневная планёрка. Загружаешь остатки и заказы за прошлую неделю — получаешь готовые заявки: какой товар, каких размеров, с какого склада и на какой везти. Прогноз = прошлая неделя × (1 + рост%).

Параметры

Формула прогноза. Если на прошлой неделе артикул получил 15 заказов в регионе — на эту неделю закладываем 15 × 1,15 = 18 шт. Если на ближайшем складе сейчас меньше 18 — делаем заявку на довоз недостающего с самого «жирного» склада-донора, распределяя по размерам пропорционально тому, что есть у донора.

Экономика ИРП

Сколько денег теряешь сейчас и сколько сэкономишь при достижении цели.

Источник карты «быстрых складов»

Встроено 102 города России/СНГ с топ-6 складов WB и временем доставки в часах. Источник — тг-бот @POSTAVLENOru_BOT, данные обновляются 3 раза в день на стороне бота (снимок на момент сборки сайта).

Для каждого региона/города сайт берёт приоритет №1 (самый быстрый склад) как целевой для перераспределения. Можно писать полное имя («Ростов-на-Дону»), сокращённо («Ростов»), либо название федерального округа («Приволжский») — сработает мягкое сопоставление.

Охватываемые склады: Коледино, Электросталь, Подольск, Белые Столбы, Алексин (Тула), Рязань, Владимир, Казань, Новосемейкино, Сарапул, Екатеринбург, Новосибирск, Краснодар, Невинномысск, Волгоград, Воронеж, СПб Шушары, Калининград, Котовск, Пенза, Истра, Белая Дача, Обухово, Чашниково, Барнаул СЦ 2, Владивосток, Перспективный + склады Казахстана/Армении.

Оговорки, которые важно знать

    
    
      
  • Сайт работает полностью на клиенте — токен никуда не отправляется, кроме самого WB API (или твоего локального прокси).
    • 
      
  • Карта быстрых складов — это снимок на момент сборки сайта. Времена доставки у WB меняются — для 100% актуальности сверяйся с ботом-источником.
    • 
      
  • КТР рассчитан по аппроксимации ДЛ → коэффициент. Точную таблицу WB не публикует, но расхождение со скриншотами кабинета обычно ±0.05.
    • 
      
  • Окно ИРП — 13 недель. Не жди результата через 2 дня.
